# Industrial Predictive Maintenance using Machine Learning

## Project Overview

This project focuses on analysing industrial machine operational data to understand failure patterns and develop a predictive maintenance workflow using machine learning techniques.

The work was developed during my Data Science internship at Infotact Solutions as part of a collaborative machine learning project.

This repository contains my individual contributions focusing on feature engineering, contextual analysis, model evaluation, and machine behaviour analysis.


## Dataset

Dataset:
AI4I 2020 Predictive Maintenance Dataset

The dataset contains industrial machine measurements including:

- Air temperature
- Process temperature
- Rotational speed
- Torque
- Tool wear
- Machine failure indicators


## Project Workflow

### 1. Data Exploration

- Dataset inspection
- Missing value analysis
- Failure distribution analysis
- Correlation analysis


### 2. Feature Engineering

Engineered additional machine behaviour features:

- Temperature difference
- Power-related features
- Wear-torque relationship features
- Operational interaction features


### 3. Context-Aware Feature Analysis

Extended the analysis by incorporating contextual variables:

- Ambient temperature
- Factory load
- Humidity

Created interaction features to study the relationship between operational conditions and machine failures.


### 4. Model Analysis

Performed machine learning analysis using:

- Random Forest
- Classification models
- Comparative evaluation


### 5. Feature Importance Analysis

Analysed important machine parameters influencing failure prediction.
